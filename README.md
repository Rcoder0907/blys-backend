# blys-backend
To run the backend follow the steps bellow.

## Clone the repository
```
git clone https://github.com/Rcoder0907/blys-backend.git backend
```

## Go to project directory
```
cd backend
```
## Install dependency
```
npm install
```

## Run the project
```
npm start
```

The appication will start listening to port 3001

## Visit url
```
http://localhost:3001
```
